## 法務・ポリシー(サイト掲載文の骨子)

### 免責事項(結果画面フッター+専用ページ)
- 本サイトの備蓄量・チェックリストは公的ガイド(出典明記)に基づく**目安**であり、安全を保証するものではない
- 実際の災害時は自治体・気象庁等の公式情報を最優先すること
- 本サイトの利用により生じた損害について、運営者は責任を負わない
- 出典: 東京備蓄ナビ(東京都)、農林水産省「災害時に備えた食品ストックガイド」、内閣府防災情報、ハザードマップポータルサイト(国土地理院)— 政府標準利用規約(第2.0版)に従い出典を明記して利用

### プライバシーポリシー
- 入力データ(家族構成・備蓄状況・集合場所等)は**ブラウザ内(localStorage)にのみ保存され、サーバーに送信されない**
- アクセス解析は Cloudflare Web Analytics(クッキー不使用・個人を特定しない)のみ
- 共有URL機能を使う場合、URLにはデータが含まれる。URLを渡した相手には内容が見える旨を明記
