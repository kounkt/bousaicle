# 06. ブランドガイド — チエロ・デザインシステムのアプリ適用

キャラクター「チエロ」のデザインシステム(キャラデザインシート)を本アプリに適用するためのガイド。**本アプリは「チエロが作ったもの」として公開する。**

## キャラクター基本情報(デザインシートより)

- 名前: チエロ / 性別: 不明 / 年齢: ひみつ / 身長: ちいさい
- 性格: クールで頭の回転がはやい。ちょっと生意気だけど、とても親しみやすい
- キーワード: **ゆるい・親しみやすい・ちょい生意気・知的**
- 口ぐせ: 「ふふん、まかせてよ。」
- 好きなこと: 考えること、スマホいじり、コーヒー(たぶん)

## カラーシステム

| 役割 | 色 | HEX | アプリでの使い方 |
|---|---|---|---|
| レッド(メイン) | 赤 | `#E60012` | プライマリボタン、優先度S、スコア強調、アクセント。**面で多用しない**(警告色と紛れるため) |
| ブラック(メイン) | 黒 | `#111111` | 本文テキスト、見出し |
| ホワイト | 白 | `#FFFFFF` | 背景ベース |
| スキン | 淡黄 | `#FFF2D6` | セクション背景、チエロ吹き出しの背景 |
| グレー(アクセント) | 灰 | `#BDBDBD` | 罫線、無効状態、補足テキスト(本文には使わない=コントラスト不足) |

追加トークン(アプリ用に拡張。デザインシートの世界観を壊さない範囲):
- `--red-dark: #B8000E`(ホバー/押下)
- `--gray-bg: #F5F5F5`(リスト行の交互背景)
- セマンティック色: 期限警告にも赤系を使うが、装飾の赤と区別するため**警告は必ずアイコン+文言を併記**(色だけに頼らない=a11y)

コントラスト注意: `#E60012` 地に白文字はコントラスト比 約4.6:1 で AA(通常テキスト)ぎりぎり可。ボタン文言は16px以上・太字とする。`#BDBDBD` の文字は大型テキスト以外使用禁止。

## ロゴ・タイトル表記

- アプリ名表記: 「**ボウサイクル**」+ 赤帽子アイコン(チエロの帽子をシンボル化)
- タグライン: 「チエロと3分、わが家の備え。」
- デザインノート(シートより): 「帽子は必ず大きく描く」「赤・黒・白を基調とする」「ちょいキラ✦やや効果で魅力UP」→ ✦(スパークル)はスコア演出・達成時のみ使用

## トーン&ボイス

チエロの声 = プロダクトの声。詳細なコピー集は [04_ux_screens.md](04_ux_screens.md)。

| する | しない |
|---|---|
| 「ふふん」と余裕を見せる(知的な安心感) | 不安を煽る、急かす、脅す |
| 短く言い切る。「水とトイレが本体」 | 長い説教、専門用語の羅列 |
| ちょい生意気(「…やるじゃん」) | 上から目線の説教、馴れ馴れしすぎる絵文字連打 |
| 失敗に寛容(「気づけたのが偉い」) | 罪悪感を使った動機づけ |
| 根拠がある自信(出典に裏打ち) | 根拠のない断定(「これで絶対安心」) |

災害発生直後の特別ルール: ✦やドヤ顔演出を自動抑制するフラグを用意(検討)。少なくともSNS発信は被害状況が見えるまで宣伝的投稿を止める。

## タイポグラフィ

- 和文: system-ui スタック(`"Hiragino Sans", "Noto Sans JP", sans-serif`)。Webフォントは読み込み速度優先で使わない(災害時の弱回線要件)
- 見出しのみ太め(700)、本文400。最小フォントサイズ16px(スマホ)
- 数字(スコア・数量)は大きく・タブラー数字で

## レイアウト原則

- デザインシートの紙面のように: 白地+黒罫線+赤ラベルの「シート感」。角丸は控えめ(8px)、黒の太枠(2px)をカードに
- チエロは「画面の主役」ではなく「余白にいる案内役」。1画面に1体まで
- ハーフトーンドット・斜めグレー帯(シート背景のモチーフ)をランディングの装飾に流用可
