# ボウサイクル(仮称)— チエロと3分、わが家の備え

> ふふん、まかせてよ。— チエロ

**「不安なのに、何をすればいいかわからない」を、3分で「やることリスト」に変える無料Webアプリ。**

登録不要・ブラウザで即使える、家庭の防災備蓄の「診断 → リスト → 継続管理(ローリングストック)」一気通貫ツール。チエロ(赤い帽子のキャラクター)が案内役を務める、チエロ作のプロダクトとして公開する。

## なぜこれを作るのか(1分サマリー)

1. **需要**: 南海トラフ臨時情報(2024年8月)を約80%が認知し70%以上が不安を感じたのに、新たに備蓄した人は26.7%。非備蓄理由のトップは「何から手を付ければいいかわからない」(約5割)。— 意識と行動のギャップが政府系調査で定量確認済み
2. **市場の空白**: 速報通知系(Yahoo!防災・NERV)は完成形。一方「家庭の備え」側は、東京備蓄ナビが「リスト生成で終わる」、備蓄管理アプリは「手入力が重く小規模」。**診断と継続管理の間に断絶がある**
3. **エビデンス**: if-thenプラン(実装意図、d=0.65)、環境省の「冷蔵庫に貼るシート」RCT(日本の1,200世帯で防災行動が有意に改善)、摩擦削減ナッジなど、機能に直接翻訳できる実証済み手法が揃っている
4. **個人開発と相性が良い**: 登録不要・localStorage・静的ホスティングで完結。運用コストほぼゼロで一般公開できる
